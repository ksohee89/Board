# Board Project
Spring Boot, jsp로 구현한 게시판 프로젝트

## 📁 프로젝트 소개
웹 사이트 개발의 기반이 되는 CRUD 기능을 게시판을 직접 혼자 구현해보며 웹 개발의 기초를 다지고자 시작하게 되었습니다.
